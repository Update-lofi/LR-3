Лабораторная работа №3: База данных для «Личного дневника»

Цель: Создать базу данных SQLite для проекта «Личный дневник» и реализовать функции для работы с записями.

Связь с предыдущими работами: Это продолжение ЛР1–ЛР2. Теперь добавляем базу данных.

Задание 1. Создайте файл database.py

Создайте файл database.py со следующими функциями:

Функция Описание
get_db_connection() Устанавливает соединение с БД, возвращает объект соединения
init_db() Создаёт таблицу entries, если её нет
add_entry(title, content) Добавляет новую запись (дата ставится автоматически)
get_all_entries() Возвращает все записи, отсортированные от новых к старым
get_entry(entry_id) Возвращает одну запись по id
update_entry(entry_id, title, content) Обновляет заголовок и текст записи
delete_entry(entry_id) Удаляет запись по id

Требования к таблице entries:

Поле Тип Ограничения
id INTEGER PRIMARY KEY AUTOINCREMENT
title TEXT NOT NULL
content TEXT NOT NULL
created_at DATE NOT NULL

Задание 2. Создайте тестовый скрипт test_db.py

Напишите скрипт, который последовательно:

  1. Инициализирует базу данных
  2. Добавляет 3 записи с разными заголовками и текстом
  3. Выводит все записи в формате: id. Заголовок (дата)
  4. Выводит одну запись с id = 2 (заголовок, содержание, дата)
  5. Обновляет запись с id = 2 (меняет заголовок и текст)
  6. Выводит обновлённую запись для проверки
  7. Удаляет запись с id = 3
  8. Выводит оставшиеся записи

Задание 3. Самостоятельная работа

Добавьте в database.py следующие функции:

Функция Что делает SQL-подсказка
get_entries_count() Возвращает количество записей SELECT COUNT(*) FROM entries
search_entries(query) Ищет записи, где заголовок содержит query SELECT * FROM entries WHERE title LIKE '%?%'
delete_all_entries() Удаляет все записи DELETE FROM entries
get_last_week_entries() Возвращает записи за последние 7 дней SELECT * FROM entries WHERE created_at >= date('now', '-7 days')
